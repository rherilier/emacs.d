# my Emacs config

## Shortcuts

* `ctrl-tab` and `ctrl-shift-tab`: swap buffers;
* `super-d l`: ui-menu;
* `super-p`: projectile shortcuts prefix;
* `super-c suber-b`: emacs configuration bootstrap;
* `super-d i`: initialize project;
* `super-d c`: reconfigure project;
* `f9`: compile;
* `f10`: recompile;
* `f11`: next error;
* `f12`: previous error;
